整数表达

1.char

char输出的是字符串。

char c;

scanf(“%c”,&c);输入：1

printf(“c=%d”,c);输出：49（当以d输出时显示的的是ASCLL码，在电脑内部字符“1”是用这个数代替）

printf(“c=%c”,c);输出：1（现在输出的是字符1）

如果是用int则：

int i;

char c;

scanf(“%d”,&i);输入49

c=i;

prinf(“c=%c”,c);输出：C=1（这时又把数字49转换成了字符1）

printf(“c=%d”,c);输出：c=49（49类型本来就是整数，直接输出）

2.如何输入1这个字符给char c?
·scanf(“%c”, &c);>1
• scan(“%d’,&i): c=i;>49
·1的ASCII编码是49，所以当c==49时，它代表’1
•printf(“%i %c\n”,c,c);
一个49各自表述!

3.有无空格的区别(在输入端有无空格的区别)

int a;

char c;

scanf(“%d %c”,&a,&c); 或者 scanf(“%d%c”,&a,&c);

printf(“a=%d c=%d c=%c”,a,c,c);

有空格指读到整数后面加空格，然后把下一个数字给下一个变量。

无空格指读完整数直接跳过所以后面的变量就是空格（如果是字母则可以给下一个变量）

（忘了就打开存c-字符计算实验几遍，没特殊需求加空格）

4.第四点也是让我最震惊的字符加减可以变成别的字符

char c=’A’;

c++;

printf(“%c\n”,c);

第一次输出：B

第二次输出：C

……….以此类推

因为A的ASCLL的编号是32，B的编号是33。

所以可以在整数和字符串之间转换。

关于C语言-复合赋值

1.复合赋值

total+=(sun+100)/2 和相同 total=total+(sun+100)/2

2.递增递减运算符

++”和“–”是两个很特殊的运算符，它们是单目运算符，这个算子还必须是变量。这两个运算符分别叫做递增和递减运算符，他们的作用就是给这个变量+1或者-1。

count++;
count +=；
count = count + 1;

++和–可以放在变量的前面，叫做前缀形式，也可以放在变量的后面，叫做后缀形式。
a++的值是a加|以前的值，而++a的值是加了1以后的值，无论哪个，a自己的值都加了1了。

按我自己的理解就是:

a=0

printf(“a++=%d”,a++)

printf(“a=%d”,a)

输出：a++=0 a=1

printf(“++a=%d”,++a)

printf(“a=%d”,a)

输出：++a=1 a=1

3.运算优先级

单目>乘除>加减>对比>赋值

运算一定要遵循运算优先级,对比如果正确则输出1反之输出2

关于C语言--变量，浮点数和变量交换

1.<类型名称> <变量名称>= <初始值>

如：int price= 0;**

​ int amount = 100;

1.1组合变量定义的时候，也可以在这个定义中单个赋值。

​ int price = 0, amount = 100; 或者 int price,amount=0;

1.2 不一定要在最初赋值但一定要在使用前赋值。C99协议变量可以在任何地方定义

  1. const

    const是一个修饰符，加在int的前面，用来给这个变量加上一个const（不变的）的属性。这个const的属性表示这个变量的值一旦初始化，就不能再修改了。

    如果你把锁定了的变量放在左边这会报错。

  2. 浮点数.

    在浮点数里scranf()中获取数值要用%lf，但用printf()依然用%f，

    3.1 double
    inch是定义为int类型的变量，如果把int换成double，我们就把它改为double类型的浮点数变量了。
    double的意思是“双”，它本来是“双精度浮点数”的第一个单词，人们用来表示浮点数类型。除了double，还有float（意思就是浮点！）表示单精度浮点数。

  3. 交换变量

    交换变量需要一个新的容器（另一个变量）。

    如：a=1 b=2 直接 a=b b=a 不能直接交换而是会顺序运算。

    正确应该引进一个新变量c

    如： a=1 b=2 引进一个c

    a=c a=b b=c 就可以完成交换。
