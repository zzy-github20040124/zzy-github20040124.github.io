类型转换-逻辑运算-变量范围

1.强制类型转换(强制转换的优先级高于四则运算)

(转换的类型)数值

double a=1.0;

double b=2.0;

int i =(int)a/b;

这里会先把a转换成int类型在除以b；

如果要先除可以int i=（int)(a/b);

2.自动类型转换

当运算符的两边出现不一致的类型时，会自动转换成
较大的类型
·大的意思是能表达的数的范围更大
char–> short–> int –> long —> long long
·int –> float –> double

对于printf，任何小于int的类型会被转换成int；float会被转换成double；

但scanf不会，要输入short，需要%hd；

3.逻辑运算

！ 逻辑非 !a 如果a是true结果就是false，如果a是false结果就是true。

&& 逻辑与 a&&b 如果a和b都是ture，结果就是ture否则就是false。

|| 逻辑或 a||b 有一个是ture就是ture否则就是false。

优先级：！>&&>||

如果要表达一个范围

x>4 && x<6就是数学中的4<x<6;

4.函数定义的范围局域在{}中

在函数中定义一个变量只能在同一个{}起作用出了大括号就不被定义了。

int main(){int a=1; int b=2;}(在这个函数中我定义了两个变量但在其他函数中就不能用)

viod sum(){a+=b;}(就会报错因为在sum函数中a和b没有被定义)

会优先使用范围更小的变量
