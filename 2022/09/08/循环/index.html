1.while函数的格式

需要执行的语句

while（判断条件）{

​ 需要执行的语句（直到条件不再符合这个判断）}

（反正在whlie函数前应该再加一个相同的执行语句，就是在进入循环前需要把被执行的语句重复一遍在进入循环）

1.1do…while

do{需要执行的语句;}while（判断条件）;

与单while的区别是不用在前面多加一个相同的执行语句。

2.for循环

for(初始动作;循环继续的条件;循环每一轮要做的事){

​ 需要执行的语句

}

3.总结

循环的计算和选择

  • 如果有固定次数，用for
  • 如果必须执行一次，用do_while
  • 其他情况用while

循环控制

  • break 跳出循环
  • continue 跳过剩下的语句进入下一轮
  • break和continue只在它所在的那层循环作用

从嵌套循环跳出

在多层循环下使用goto

操作：层内goto out; 配合层外out:使用，然后加上return 0;

  • 给定不超过6的正整数，考虑从a开始的连续4个数字，输出所有能用它们组成的无重复的3位数

  • 格式：从小到大，每行6个整数，中间用空格隔开 但行末不能有多余空格

  • 样例：输入2，输出234,235,243,245…

  • 思路：需要3位数，也就是3重循环，每一位都可能是a,a+1,a+2,a+3

  • 3重循环 如果满足i,j,k互不相等的条件，则输出3位数

  • 定义count用于记录次数，count==6则换行，然后count值清零进入下次循环，如果count不等于6，else输出空格

注意事项：

  • 编程语言里0就是false，非0就是true，在！面前只有true和false
  • if 为false时候 大括号里的语句不执行
  • 当 while 为 True 时运行【语句块】，【语句块】运行结束后，再次进入 进行 判断，如果 结果为 True 则再次运行【语句块】, 以此循环直到 结果为 False 结束循环。
