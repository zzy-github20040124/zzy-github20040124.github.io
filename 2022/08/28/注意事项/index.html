hexo的困难

​ 第一次搭建博客真的遇见了好多困难，一次次的报错真的叫人崩溃，还好现在成熟的网络想找到答案也不是一件困难事。

1.fatal port 4000 has been used.这个代表你用的域名你现在还不能用。

可以等到一个可以用的时机或者换一个域名(使用命令hexo server -p 加上你想更改的端口)

2.can not read a block mapping.（这是一个很小的错误但是我们往往会忘记）

在配置文件里面，都是按照属性名+英文冒号+英文空格+属性值，这个形式进行属性设置的，所以千万不要忘记空格不然报错真的很心累。

（我确实也没想到hexo能使用HTML中的语法，倒是很好用）
