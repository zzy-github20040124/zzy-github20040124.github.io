java

Java是面向对象的语言学了C语言后学习Java还算轻松，学到面向对象做笔记记录一下。

1.java所有的操作都在类里（目前为止），类的关键词class是主要。

public class 类名（加了public的类要与文件名一样）{

​ public static void main(){

​ 具体方法；

​ }

}
与C语言不一样的是C语言只有main和其他函数，而java在不同的类中可以有很多函数。类似于结构体但java可以包含函数及属性。

2.函数输出：System.out.println(…) 和System.out.print(….)区别在于是否换行。可额外加/n换行。

函数输入：先要导包类似于C语言头文件一样的。import java.util.Scanner

然后把这个包赋给一个值类似于 Scanner scan=new Scanner(System.in)

然后就可以用scan从键盘上读入了类似：int n=scan.nextInt()

3.类型；char字符只能读一个，String字符串可以读很多，int整形，float浮点。类型问题需要特别注意比如：short i=n+1这是不行的因为1会被认定成int型。浮点型也是默认为double而不是float；

4.面向对象class出一个新的类

类就像抽象的-

class 类名{

​ 可以存放属性如：

​ int number；

​ 可以创建一个方法

​ public void do(可以放形参){

​ 写方法；

​ }

}

（1）：创建类，设计类的成员

（2）：创建类的对象

（3）：通过“对象.属性”或者“对象.方法 ”来调用对象的结构。

比如在main里 类名 变量名 = new 类名（）

这个变量名就是对象可以通过它来带调用；
